## 📝 About The Game
**My Voxel Game** is an enhanced sandbox based on the famous **Mineclonia** game and powered by the open-source **Luanti (formerly Minetest)** engine.

Unlike the vanilla version, this version comes with essential mods **pre-installed and integrated** directly into the core gameplay, so you can play instantly without downloading anything from the store.

### ✨ New Integrated Features:
*   **🏠 Interactive Furniture Mod:** Everything works! Fridges, ovens, chairs you can sit on, tables, lamps, and hundreds of decoration items.
*   **🏚️ Structures & Dungeons Mod:** Your world is no longer empty! Automatically generated caves, abandoned houses, railways, traps, resources, villages, tunnels, mines and more.

Plus all the original features of Mineclonia!

## 🎮 Default Controls
| Key | Action |
| :--- | :--- |
| **W, A, S, D** | Movement |
| **Space / Shift** | Jump / Sneak |
| **Left Mouse Button** | Dig / Punch |
| **Right Mouse Button** | Place / Use |
| **I** | Inventory |
| **T** | Chat |
| **Esc** | Pause Menu |
