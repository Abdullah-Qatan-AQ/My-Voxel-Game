## 📝 About the Game
**My Voxel Game** is an exciting and fun sandbox building and exploration game powered by the open-source voxel engine **Luanti (formerly Minetest)**. The game allows you to easily install various mods and add-ons directly from the built-in in-game content store to fully customize your world and gameplay experience!

---

## 🎮 Default Controls
You can easily customize all of these keybindings in the game settings at any time:

| Key | Action |
| :--- | :--- |
| **Move Mouse** | Look around / Change camera direction |
| **W, A, S, D** | Movement (Forward, Left, Backward, Right) |
| **Space** | Jump / Move up |
| **Shift** | Sneak / Move down |
| **Q** | Drop itemstack |
| **Shift + Q** | Drop a single item |
| **Left Mouse Button** | Dig / Punch / Use |
| **Right Mouse Button** | Place / Use |
| **I** | Open Inventory / Character menu |
| **Z** | Zoom (Requires zoom privilege) |
| **T** | Open Chat window |
| **Esc** | Pause menu / Abort / Exit |

---

## 📁 Important Paths
If you are modifying the game files or transferring your data:
* **Worlds:** All of your created worlds can be found inside the `user/worlds/` directory.
* **Settings:** The main configuration file is `user/minetest.conf`. It is automatically generated after closing the game for the first time.
